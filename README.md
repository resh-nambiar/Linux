# Linux Documentation

## Overview
Linux is a free and open-source operating system used to run computers, servers, and cloud platforms.  
It is known for its security, speed, and reliability.

This repository contains **basic information and commands of Linux**.  
The **detailed documentation is provided in the uploaded document file**.

---

## What is Linux?
Linux is an operating system that manages hardware and software resources of a computer.

---

## Why Linux?
- Free and open-source
- Secure and stable
- Widely used in servers and cloud
- Preferred by developers
